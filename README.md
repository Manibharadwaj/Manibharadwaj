### what i do

i build things that **do things on their own**. ai agents that call people. bots that apply to jobs. scrapers that find leads. hardware that reads the air you breathe. and the occasional clone just because.

```
python → the glue for everything weird i build
typescript → when it needs to be a real product
dart/flutter → mobile apps that talk to ai
swift → ios experiments
c → when the hardware demands respect
```
